                IBM 4765 PCIe Cryptographic Coprocessor Toolkit

                          Skeleton Server Sample 

___________
Description

The code in this sample shows how to create a multi-threaded card-side 
application.  Each of the threads spawned on the card service a specific
type of request, such as PKA, RNG, DES, or AES.

___________
Build Notes

For information on how to build and execute this sample, please refer to The 
IBM 4765 PCIe Cryptographic Coprocessor Custom Software Toolkit Developer's 
Guide located in y4tk/<version>/docs


_____________
Special Notes

This sample is a good place to look for examples of how to program various API
calls.
